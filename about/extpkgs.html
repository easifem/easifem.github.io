External pkgs

For some functionality EASIFEM depends upon FORTRAN packages developed by the others. The table given below briefly describes about the external packages which have been used in EASIFEM. You can find more information about the external packages here.

❤️ 🙌

The developers of EASIFEM appreciate the creators of these external packages as these packages have eased the development of EASIFEM.

note

Some external packages shipped with the EASIFEM as they have been modified according to the need of EASIFEM. All these packages are meant to be used within EASIFEM, if you are interested in any of these external packages, then please refer to the original and official documentation of these external packages.

Pkg nameDescriptionCommand
OpenBLAS Highly optimized BLAS 🚀 More... easifem install openblas
SuperLU Direct solution of large, sparse, nonsymmetric systems of linear equations 🚀 More... easifem install superlu
LIS Linear interative solver 🚀 More... easifem install lis
METIS Mesh and graph partitioning library 🚀 More... easifem install metis
SCOTCH Mesh partitioning library 🚀 More... easifem install scotch
ARPACK Eigensolver for sparse matrices 🚀 More... easifem install arpack
FFTW Fast Fourier Transform library. 🚀 More... easifem install fftw
GTK-Fortran Fortran bindings for GTK-4 library 🚀 More... easifem install gtk-fortran
LAPACK95 Fortran 95 interface for LAPACK library 🚀 More... easifem install lapack95
Sparsekit Fortran library for linear algebra on sparse matrices 🚀 More... easifem install sparsekit
Gmsh Finite element mesh generator 🚀 More... easifem install gmsh

Click on 🚀 icon to get the information about how a package has been incorporated in the EASIFEM.

easifemBase

EASIFEM is a Modern Fortran framework (suite of scientific libraries) that facilitates the Expandable And Scalable Infrastructure for Finite Element Methods. EASIFEM “eases” the efforts to develop scientific programs in FORTRAN for solving partial differential equations (PDEs) using finite element methods and other grid-based methods. It is meant for researchers, scientists, and engineers.
