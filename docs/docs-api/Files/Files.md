# Files

Files are designed for IO operations. Following file formats are supported in EASIFEM.

- [[FortranFile_]]
- [[TxtFile_]]
- [[InputFile_]]
- [[HDF5File_]]
- [[XMLFile_]]
- [[VTKFile_]]

At the root we have [[AbstractFile_]] which is an abstract type for files.
