In this section presents the details of using EASIFEM in your project. There are several ways to use `easifem` in your development environment. Please select the one which is most suitable for you.

## `easifem` CLI (recommended)

The easiest way to use `easifem` is by using `easifem` command line interface. You can obtain the command line interface by using the details presented in [previous sections](/guides/install). 

## CMake

As we have mentioned before `easifem` uses CMake build system. Integrating your project with `easifem` is very simple.
